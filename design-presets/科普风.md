# 科普风 / Science-Explainer Style

## Palette

- Background: `#050308` deep night, `#0a0612` to `#02010a` gradient
- Primary text: `#f4ead0` cream
- Gold accent: `#d4a64a` solid, `#f4d479 → #c98f2e` highlight gradient
- Purple accent: `#3a1d6d` (used sparingly)
- Data colors: `#f4d479` / `#5fc4f4` / `#9b6cff` / `#ff8c5a` (sequential, distinguishable on dark)

## Typography

- Sans (default): system / "PingFang SC" — narration, captions
- Display: weight 500, letter-spacing 0.04em — section titles
- Mono: `ui-monospace` — numbers, timestamps

## Motion

- Eases (mandatory non-linear): `power3.inOut`, `sine.inOut`, `expo.inOut` (mix 3+)
- Stagger: 80-140 ms between siblings
- Camera intensity bias: **subtle** — this is information, not drama
- Pulse / breath only on hero phrases, never on data

## Method preference

- Data-heavy content → prefer `rm-d3-bar-chart` / `rm-d3-line-trend` (numbers matter)
- Pull-quotes / emphasis → `hf-waapi-marker` (highlight key fact)
- Section transitions → use `fade` (gentle, doesn't break flow)
- Avoid `dip-to-black` (feels too cinematic for explainer pace)

## Voice

- Default voice: `presenter_male` (calm, podcast)
- Use `audiobook_male_1` for emotional / personal anecdotes
- Use `presenter_female` for contrasting voice / "the other side said…"
- Aim for 2-3 voice changes per 60 s

## Don'ts

- No `Math.random()` / `Date.now()` (deterministic)
- No `repeat: -1` on timelines
- No full-screen gradients on dark bg (banding)
- No drama eases on chart scenes (data should feel solid)
