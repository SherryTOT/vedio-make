# 商业风 / Business / Corporate · 叙事调性档

> 拷成项目根 `design.md` 使用。只写节奏 / 语气 / 观众 / 风格档，**不写色值 / 字体 / 光效**。
> 视觉真相在 `src/methods/designs.ts` tokens，动效在 `docs/MOTION.md`，选法在 `docs/DIRECTION.md`。

## 设计预设（视觉血统）

- 预设 id：`swiss`（极简黑白，权威、精确、决断）
  - 想要蓝色强调 + 巨号衬线的印刷气场可切 `magazine`。
- 强调色一律来自预设 token，别在文案里指定颜色。

## 风格档案（DIRECTION §二）

- 主档：**A 极客湾式**（指标 / 定价 / 性能对比）。
- 品牌主张、发布金句可借 **C 印刷工坊经典**。

## 节奏密度

- 目标：每 **2.5–3s** 一个视觉事件，比科普更紧。
- tempo：`snappy`（自信、决断，不拖泥带水）。

## 语气 / 目标观众

- 观众：决策者 / 专业受众，时间宝贵。
- 语气：笃定、清楚、有信息增量；一句一个卖点，不绕。

## 方法偏好（语义，落到 DIRECTION §一 映射表）

- 定价 / 指标 → `hf-mega-counter` / `hf-versus-panel` / `rm-d3-bar-race`。
- 功能清单 → `hf-anime-scatter`（逐项弹入）。
- 主张金句 → `hf-word-punch` / `hf-kinetic-text`（强调靠变色+上浮，不靠发光）。
- 关系 / 流程 / 钱怎么流 → `hf-scribble-annotate` 箭头流。

## 配音

- 默认解说：`male-qn-jingying`（精英男，干脆）。
- 客户引述「他们说……」用 `presenter_female`。
- 两把声音足够，别为花哨换声。

## Don'ts（继承印刷工坊命门）

- 进场禁 bounce / back.out（商业语境显幼稚），过冲最多 1 次 ≤8%。
- 数据镜禁 kenburns；禁渐变 / 发光 / 投影 / AI 金光紫气（MOTION.md §一红线）。
- 动效参数看 `docs/MOTION.md`，色值/字体看 design tokens——都不在这里写。
