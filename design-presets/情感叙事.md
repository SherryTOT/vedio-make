# 情感叙事 / Emotional Narrative · 叙事调性档

> 拷成项目根 `design.md` 使用。只写节奏 / 语气 / 观众 / 风格档，**不写色值 / 字体 / 光效**。
> 视觉真相在 `src/methods/designs.ts` tokens，动效在 `docs/MOTION.md`，选法在 `docs/DIRECTION.md`。

## 设计预设（视觉血统）

- 预设 id：`claywarm`（暖手作，温暖、亲和、有人味）
  - 想要更克制的信笺 / 衬线气质可切 `inkwork`。
- 情绪段落全程一个预设，别在段内跳档。

## 风格档案（DIRECTION §二）

- 主档：**B 小Lin说式**拼贴叙事——讲人、讲事、讲关系。
- 情绪高点、片尾定帧可借 **C 印刷工坊经典**。

## 节奏密度

- 目标：每 **3–4s** 一个视觉事件，允许长镜久留（情绪需要呼吸）。
- tempo：`gentle`（从容、有机），别用锐利的快节奏切。

## 语气 / 目标观众

- 观众：愿意被带入、听故事的中文观众。
- 语气：口语、真诚、有温度；衬线大字承载情绪金句，关键词弹字承担笑点/泪点节拍。

## 方法偏好（语义，落到 DIRECTION §一 映射表）

- 情绪金句 → `hf-word-punch` / `hf-kinetic-text`（衬线、不弹跳）。
- 人物 / 公司 / 物件登场 → `hf-sticker-pop`（生图白底抠像贴纸）。
- 照片 / 实拍 → `rm-image-kenburns` + spotlight 聚焦人脸。
- 章节切换 → `hf-chapter-card`；关系指向 → `hf-scribble-annotate` 手绘箭头。
- 数据图表**降权**——这类片子不靠数字（硬要用别打断情绪）。

## 配音

- 默认解说：`audiobook_male_1`（低沉、有思绪）或 `audiobook_female_1`。
- 引文 / 换人称 → `female-chengshu`（成熟温暖）或 `male-qn-qingse`。
- 换声标记章节 / 时间跳转；每 90s 约 3–5 次换声。

## Don'ts（继承印刷工坊命门）

- 禁锐利进场（back.out）、禁过紧 stagger（像在戳观众）。
- 禁渐变 / 发光 / 投影 / AI 金光紫气 / 高饱和堆色（MOTION.md §一红线）。
- 动效参数看 `docs/MOTION.md`，色值/字体看 design tokens——都不在这里写。
